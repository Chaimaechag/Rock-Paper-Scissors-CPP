# 🎮 Rock, Paper, Scissors Game (C++)

### 📝 Description
A classic "Rock, Paper, Scissors" game built with **C++**. This console application allows a user to play against a computer with randomized choices. It tracks round wins, losses, and draws to determine an ultimate champion.

### 🚀 Key Features
* **Round System:** Users can specify the number of rounds (1-10).
* **Dynamic Feedback:** The console UI changes colors based on the outcome (Green for Win, Red for Loss).
* **Robust Logic:** Uses `Enums` and `Structs` to maintain clean data flow and readable code.
* **Final Statistics:** Displays a detailed summary after all rounds are completed.

### 🛠️ Technologies & Concepts Used
* **Language:** C++
* **Concepts:** Enums, Structs, Random Number Generation, Loops (for/do-while), Switch-Case Logic, Function Passing.

### 📖 Learning Outcomes
Building this project helped me master **Structured Programming**. I focused on:
1. Organizing complex data into `Structs` to pass information efficiently between functions.
2. Implementing decision-making logic through modular functions.
3. Managing state across multiple game rounds using local accumulators and final result wrappers.
